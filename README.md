## Overview

This Python script scrapes popular Spotify song + artist data from [kworb.net](https://kworb.net) and Spotify album photos from [Spotify's oEmbed API](https://developer.spotify.com/documentation/embeds/tutorials/using-the-oembed-api) and stores/updates data in a PostgreSQL database.

The script will first extract artist names and Spotify IDs from kworb.net's top artists on Spotify listing page. Then for each artist, it will fetch their top songs page to extract song titles and stream counts, retrieve album cover URLs from Spotify's oEmbed API, and finally upsert all song data to the database in batches of 100 artists.

This script was built to support web-based guessing game, streamdle. See the [streamdle](https://github.com/ct0ng/streamdle) repository for more information.

## Getting Started
### Prerequisites
- Python 3.x
- Dependencies listed in `requirements.txt`
- A Supabase account and project (required if setting up alongside with the [streamdle](https://github.com/ct0ng/streamdle) app)
- A PostgreSQL database with the following schema (can be set up through Supabase):

```sql
create table public.artist (
  artist_id serial not null,
  name text not null,
  spotify_id text null,
  constraint artist_pkey primary key (artist_id),
  constraint artist_name_key unique (name),
  constraint artist_spotify_id_key unique (spotify_id)
) TABLESPACE pg_default;

create table public.song (
  song_id bigint generated by default as identity not null,
  name text not null,
  stream_count bigint not null,
  created_datetime timestamp without time zone not null default now(),
  updated_datetime timestamp without time zone null,
  artist_id integer null,
  spotify_track_id text null,
  album_cover_url text null,
  constraint song_pkey primary key (song_id),
  constraint song_title_artist_id_key unique (name, artist_id),
  constraint fk_artist foreign KEY (artist_id) references artist (artist_id)
) TABLESPACE pg_default;
```

You'll also need to create a PostgreSQL function to fetch random songs. In your Supabase SQL editor, run:

```sql
CREATE OR REPLACE FUNCTION public.get_random_songs()
 RETURNS TABLE(song_id bigint, song_name text, stream_count bigint, artist_name text, album_cover_url text)
 LANGUAGE sql
AS $function$select
    s.song_id,
    s.name as song_name,
    s.stream_count,
    ar.name as artist_name,
    s.album_cover_url
  from song s
  join artist ar on s.artist_id = ar.artist_id
  order by random()
  limit 10;$function$
```

### Installation
1. Clone the repo
```sh
  git clone https://github.com/ct0ng/streamdle-song-scraper.git
```

2. Install Python Dependencies
```sh
  pip install -r requirements.txt
```

3. Add Environment Variables  
   Create an `.env` file in the root directory of the project that will store your PostgreSQL database credentials. The file should look something like:
```
# PostgreSQL db config
DB_HOST=your-db-host.supabase.co
DB_NAME=your_database_name
DB_PORT=5432
DB_USER=your_db_user
DB_PASSWORD=your_db_password
```

## Usage
To run the script:
1. Open a terminal/commmand prompt
2. Navigate to the directory containing the script
3. Run the script
```sh
python main.py
```

Logs for this script will be written to `scraper.log` in the root directory of the project.
